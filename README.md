# Piggy Bank Smart Contract

This project is a simple Piggy Bank smart contract built as part of the Build & Learn Week challenge.

## What it does
- Allows anyone to deposit ETH
- Only the owner can withdraw ETH
- Stores balance on-chain
- Prevents unauthorized access

## Smart Contract Features
- State variables
- Access control using modifiers
- Input validation using require()
- Event logging
- Secure ETH transfers

## Tech Stack
- Solidity
- Ethereum
- Remix IDE

## How to Use
1. Deploy the contract
2. Call deposit() with ETH
3. Owner can withdraw funds

## Security
- Owner-only withdrawal
- No zero-value deposits
- Balance validation
